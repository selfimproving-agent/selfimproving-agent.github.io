Self-Improvement in Foundation Model Based Agentic Systems

A survey website designed for fast understanding of the field: its taxonomy, representative papers, and the relationship between foundation model improvement and scaffolding improvement.

Browse Papers View Taxonomy GitHub List BibTeX

Abstract

Foundation models have greatly expanded the capability of agentic systems, but building agents that can improve themselves over time remains a central challenge. This survey organizes the area into two complementary axes: foundation model improvement, which updates the model itself, and scaffolding improvement, which updates prompts, memory, tools, and higher-level agent workflows. The website is structured so readers can quickly understand the field and directly jump to representative papers under each category.

How to use this page

  1. 1. Read the taxonomy to understand the major components of the field.
  2. 2. Jump to any subsection and browse the selected papers directly on the page.
  3. 3. Use the GitHub list for the full bibliography and ongoing updates.

Key Messages

Two major loops

The survey separates slow, parameter-level improvement from fast, scaffold-level improvement.

Taxonomy-first reading

Readers can understand the field by moving from category to category rather than reading linearly.

Paper list as the core asset

The website directly embeds selected papers instead of leaving the literature only in an external repository.

Built for fast scanning

A cleaner layout reduces visual noise and helps readers quickly locate representative work.
